## Project Kelompok 2 Matkul Fundamental AI: CHATBOT PERPUSTAKAAN SEDERHANA

Project ini berbassis python scikit-learn dimana Chatbot akan merespon pengguna terkait topik perpustakaan ini

## Cara Menjalankan Project

1. Extract file zip
2. Buka folder di vscode, lalu buka terminal (tekan "CTRL + ` ")
3. Buat virtual environment (terminal):
   python -m venv venv
4. Aktifkan venv (TERMINAL):
   - Windows: venv\Scripts\activate
   - Mac/Linux: source venv/bin/activate
5. Install semua library yang dibutuhkan (terminal):
   pip install -r requirements.txt

## 5 langkah diatas hanya dijalankan sekali guna install venv(library package) untuk project ini, setelah itu lakukan hanya 2 langkah dibawah untuk menjalankan app 
6. Jalankan aplikasi(terminal):
   python app.py
7. Buka browser ke: http://127.0.0.1:5000


## JIKA ADA PENAMBAHAN PATTERN PADA JSON
 Restart ulang server, hapus terminal dan ketik ulang "python app.py" pada terminal yg baru (PASTIKAN VENV AKTIF!! VENV AKTIF AKAN TERLIHAT PADA TERMINAL DGN WARNA TEXT HIJAU JIKA TIDAK MAKA AKTIFKAN)
